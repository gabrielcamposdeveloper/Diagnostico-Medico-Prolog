# Medical Diagnosis with Prolog

A simple academic project that demonstrates how **Prolog** can be used to build a rule-based medical diagnosis system. The application combines **Prolog's logical inference engine** with a **Python interface**, allowing users to interact with the expert system through a graphical interface.

The project showcases the fundamentals of **knowledge representation**, **logic programming**, and **expert systems**, making it a great learning resource for students interested in Artificial Intelligence and Prolog.

## 🚀 Requirements

### Prolog

1. Install a Prolog compiler/interpreter.
   - Recommended: **SWI-Prolog**
   - Download: https://www.swi-prolog.org/download/devel

### Python

1. Install **Python 3**.
2. Install **PySwip** (Python interface for Prolog):

```bash
pip install pyswip
```

3. Install **PySimpleGUI** (Graphical User Interface):

```bash
pip install PySimpleGUI
```

## ▶️ Running the Project

After installing all dependencies, run the main Python application:

```bash
python main.py
```

> **Note:** Make sure SWI-Prolog is correctly installed and available in your system's PATH before running the application.
